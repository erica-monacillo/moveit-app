# Cost-Benefit Analysis

## Project

MoveIt App

## Purpose

This document estimates the development cost, operational cost, benefits, return on investment, and recommendation for the MoveIt App.

## Assumptions

| Item | Value |
|---|---:|
| Development hours | 20 hours |
| Hourly rate | 25/hour |
| Monthly hosting cost | 0 |
| Monthly maintenance time | 2 hours/month |
| Maintenance hourly rate | 25/hour |
| Estimated monthly benefit | 150/month |
| Evaluation period | 6 months |

## Development Cost

Formula:

Dev cost = total hours × hourly rate

Calculation:

20 hours × 25/hour = 500

**Total development cost: 500**

## Operational Cost

Operational cost includes hosting and maintenance.

### Hosting Cost

The project uses GitHub Pages for deployment.

GitHub Pages hosting cost for this demo project:

0/month

### Maintenance Cost

Formula:

Maintenance cost = monthly maintenance hours × hourly rate × number of months

Calculation:

2 hours/month × 25/hour × 6 months = $300

**Total operational cost for 6 months: $300**

## Total Cost

Formula:

Total cost = development cost + operational cost

Calculation:

500 + 300 = 800

**Total estimated cost: $800**

## Tangible Benefits

| Benefit | Estimated Value |
|---|---:|
| Time saved from clearer documentation | 50/month |
| Reduced errors from checklists and CI/CD | 40/month |
| Faster deployment process | 35/month |
| Less rework from improved planning | 25/month |

Estimated monthly tangible benefit:

50 + 40 + 35 + 25 = 150/month

For 6 months:

150 × 6 = 900

**Total tangible benefits: $900**

## Intangible Benefits

- Improved developer confidence
- Better project organization
- Easier maintenance
- Better documentation quality
- More professional workflow
- Better understanding of CI/CD, security, ethics, KPIs, and deployment
- Improved readiness for future software projects

## ROI Calculation

Formula:

ROI = ((Total Benefits - Total Costs) / Total Costs) × 100

Calculation:

((900 - 800) / 800) × 100 = 12.5%

**Estimated ROI: 12.5%**

## Case Study

If this system is used by 10 student developers, the impact would include improved workflow consistency, fewer missed documentation steps, and faster project review. Even if each user saves only 1 hour per month, the group could save 10 total hours monthly. At an estimated 25/hour value, that equals 250 in monthly productivity value.

## Recommendation

**Recommendation: Proceed**

The project should proceed because the estimated benefits are greater than the estimated costs over the 6-month period. The ROI is positive at 12.5%, and the intangible benefits are valuable for a student software project.

Even though the financial return is modest, the project provides strong educational value, better documentation, improved deployment practices, basic security improvements, and a more professional development workflow.
