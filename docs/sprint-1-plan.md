# Sprint 1 Plan

## Sprint Goal
To allow users to register, log in, view available drivers, see fare estimates, and book a basic ride.

## Duration
2 weeks

## Selected User Stories (with Owners)

1. User Registration  
   - Owner: Rodrigo  
   - Tasks:
     - Create backend API for registration  
     - Create frontend form  
     - Validate inputs  

2. User Login  
   - Owner: Luke  
   - Tasks:
     - Create login UI  
     - Connect to backend authentication  
     - Handle login errors  

3. View Available Drivers  
   - Owner: Erica  
   - Tasks:
     - Display list of drivers  
     - Fetch data from backend  
     - Show driver details  

4. Fare Estimate  
   - Owner: Rodrigo  
   - Tasks:
     - Implement fare calculation logic  
     - Display estimated price  
     - Connect to booking inputs  

5. Book a Ride  
   - Owner: Luke  
   - Tasks:
     - Create booking UI  
     - Send booking request to backend  
     - Confirm booking success
