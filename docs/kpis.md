# KPIs

## Project

MoveIt App

## Purpose

This document defines the key performance indicators used to monitor project quality, delivery, reliability, and maintainability.

## KPI List

| KPI | Description | Target |
|---|---|---|
| Defect Rate | Number of known defects or failed checks found during review/testing | 0 critical defects |
| Lead Time | Time from starting a task branch to merging into `dev` | 1-3 days |
| Deployment Frequency | Number of successful deployments completed during the project | At least 1 deployment per release activity |
| Response Time | Time it takes for the deployed homepage to respond | Under 2 seconds |
| Availability | Whether the deployed site is reachable online | 99% target for demo availability |
| Dependency Health | Number of known dependency vulnerabilities from audit results | 0 high or critical vulnerabilities |
| Documentation Completeness | Required documentation files completed for each week | 100% required docs completed |
