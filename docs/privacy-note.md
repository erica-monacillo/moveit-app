# Privacy Note

## Project

MoveIt App

## Data Collected

This demo project does not intentionally collect, store, or sell personal user data.

If future versions include forms, login, analytics, or contact features, the project may temporarily process basic information such as:

- Name
- Email address
- Browser or device information
- Basic usage information

## Why Data Would Be Used

Any data collected in a future version should only be used for:

- Providing the app feature requested by the user
- Debugging technical issues
- Improving reliability and user experience
- Responding to user support requests

## Data Retention

For this demo version, no user data is intentionally stored.

If data storage is added later, retention should be limited to the shortest reasonable period needed for the feature or support request.

## User Rights

Users should be able to:

- Ask what data is collected
- Request correction of inaccurate data
- Request deletion of stored data
- Ask how their data is used
- Stop using the app if they do not agree with the privacy practices

## Sensitive Data

The project should not collect sensitive information such as passwords, financial details, health information, government ID numbers, or private personal records.

## Security Practices

The project should protect privacy by:

- Avoiding unnecessary data collection
- Using environment variables for secrets
- Keeping `.env` files out of Git
- Running dependency audits
- Validating user input
