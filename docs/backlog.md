# Product Backlog

## User Stories

1. As a user, I want to register an account so I can use the app.
   - Priority: High
   - Story Points: 3
   - Acceptance Criteria:
     - User can input name, email, password
     - System validates inputs
     - Account is created successfully

2. As a user, I want to log in so I can access my account.
   - Priority: High
   - Story Points: 2
   - Acceptance Criteria:
     - User enters correct email and password
     - System authenticates user
     - User is redirected to dashboard

3. As a user, I want to book a ride so I can travel to my destination.
   - Priority: High
   - Story Points: 5
   - Acceptance Criteria:
     - User inputs pickup and drop-off location
     - System shows available drivers
     - Booking is confirmed

4. As a user, I want to view available drivers so I can choose a ride.
   - Priority: High
   - Story Points: 3
   - Acceptance Criteria:
     - System displays nearby drivers
     - Driver details are visible

5. As a user, I want to track my driver so I know their location.
   - Priority: Medium
   - Story Points: 5
   - Acceptance Criteria:
     - Driver location updates in real-time
     - Map displays driver movement
